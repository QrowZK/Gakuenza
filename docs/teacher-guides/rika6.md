# 理科 6年 — 先生向けガイド（module key: rika6）

## 1. 概要

`rika6` は、小学校6年生の理科（東京書籍「新編 新しい理科」6年に準拠した単元構成）を対象とした、児童が自分のペースで取り組める演習モジュールです。11の単元について、選択式問題（choice）と順序並べ替え問題（order）を中心に出題し、一部の単元では自動生成問題（gen）も加わり、繰り返し取り組んでも問題のバリエーションが出るようになっています。各単元には短い解説（リード文＋要点まとめ）も収録されており、演習前後の振り返りに使えます。

本モジュールは、あくまで**授業の補助・定着用の教材**です。実験・観察を伴う理科の学習指導そのものを代替するものではなく、教室での指導・実験・話し合いを行った後の知識確認や、家庭学習での復習として活用することを想定しています。

## 4. 年間指導計画（めやす）

## 5. 使い方

- **割り当て**: 「成績簿（gradebook）」の課題割り当て画面（`hub/gradebook/assign.html`）から、担当クラスに `rika6` を割り当てます。`focus_units` で単元を絞り込むと、その単元の問題がメニュー上で優先表示されます（現在授業で扱っている単元に合わせて絞り込むのがおすすめです）。
- **取り組み方**: 児童は各自のペースで、割り当てられた単元の問題に取り組みます。教室での実験・観察・話し合いを終えた後の振り返りや、家庭学習での復習用として使うと効果的です。
- **運用のヒント**:
  1. 単元の授業が一区切りついたタイミングで `focus_units` を該当単元に切り替えると、児童は「今習っている内容」に集中して演習できます。
  2. 学期末は `focus_units` の絞り込みを外し、既習単元全体から出題させることで、総復習・定着確認として活用できます。
