# 漢字検定 5級 — 先生向けガイド (module key: kanken5)

## 1. 概要

`kanken5` は、日本漢字能力検定「5級」（小学校6年生修了程度・対象漢字
1,026字）の出題範囲にそった漢字学習アプリです。5級の対象漢字1,026字
（小1〜小6の学年別漢字配当表の累計）のうち、本級で新たに学ぶ層である
小学6年生配当漢字181字を中心に据えつつ、既習の学年配当漢字も含めた
10種類の出題形式で、読み・書き取り・部首・画数・熟語知識などを幅広く
練習します。問題はすべて自動生成され、同じ問題を使い切ることがない
ため、繰り返し練習に向いています。

本モジュールは、日本漢字能力検定協会の過去問題を一切使用しない独自
コンテンツで構成されています。出題の範囲・難易度は同協会が公表する
5級の審査基準に合わせていますが、あくまで**授業の補助・定着用の自習
教材**であり、通常の漢字学習の授業や漢字ドリルに代わるものではあり
ません。検定そのものの合否を保証するものでもなく、学級での指導と
組み合わせて使うことを想定しています。

対象学年の目安：**小学6年生**（既習の漢字の復習・定着を含むため、
6年生のどの時期でも使用可能）。

## 2. 単元一覧

`kanken5` には教科書単元のような区分はなく、以下の**10種類の出題
形式（カテゴリー）**が用意されています。各形式は独立して選んで練習
でき、「まとめてれんしゅう」ですべての形式からランダムに10問出題する
こともできます。

| # | 出題形式 | 内容 |
|---|---|---|
| 1 | 読み | 熟語の読み方を答える |
| 2 | 書き取り | かなを正しい漢字で書く |
| 3 | 部首・部首名 | 漢字の部首とその名前を答える |
| 4 | 画数 | 漢字の総画数を答える |
| 5 | 送り仮名 | 送り仮名を正しくつける |
| 6 | 対義語・類義語 | 反対の意味・似た意味の熟語を選ぶ |
| 7 | 同音・同訓異字 | 同じ読みで意味の違う漢字を選ぶ |
| 8 | 誤字訂正 | 文中の誤って使われた漢字を正しく直す |
| 9 | 四字熟語 | 四字熟語の空欄に入る漢字を答える |
| 10 | 熟語の構成 | 二字熟語の組み立て方（似た意味を重ねる・反対語を重ねる等）を見分ける |

各形式は小学6年生配当漢字181字を中心とした独自の熟語・四字熟語・
対義語類義語・送り仮名・同音同訓異字・部首・誤字訂正データから、
毎回異なる問題として自動生成されます。

## 3. 学習指導要領との対応（めやす）

> **※ めやす（要確認）** — 以下は本モジュールの範囲を学習指導要領・
> 学年別漢字配当表と照らした暫定的な対応表であり、正式な採用・教材化
> の前にカリキュラム担当者による確認が必要です。

本モジュールは学習指導要領の特定の単元に対応するものではなく、
「漢字検定5級 対象漢字1,026字（小1〜小6の学年別漢字配当表の累計）」
という級の出題範囲全体を対象とします。国語科の学習指導要領における
該当箇所は以下のとおりです。

- **〔知識及び技能〕(1)ウ 漢字** — 「第4学年までに配当されている
  漢字に加え、第5学年及び第6学年の各学年においては、当該学年の前の
  学年までに配当されている漢字を書き、文や文章の中で使うとともに、
  第6学年においては、第5学年までに配当されている漢字に加え、当該
  学年に配当されている漢字を読むこと。また、第6学年においては、
  学年別漢字配当表に示す漢字を書き、文や文章の中で使うこと。」
- 上記の学年別漢字配当表の**小1〜小6の累計1,026字**が5級の対象漢字
  であり、うち小学6年生配当の181字が本モジュールの新出中心。

出題形式ごとの対応（めやす）は以下のとおりです。

| 出題形式 | 対応する指導要領上の内容（めやす） |
|---|---|
| 読み・書き取り | 〔知識及び技能〕(1)ウ 漢字の読み書き |
| 部首・部首名 | 〔知識及び技能〕(1)ウ 漢字の構成についての理解（部首） |
| 画数 | 〔知識及び技能〕(1)ウ 漢字の筆順・画数の理解 |
| 送り仮名 | 〔知識及び技能〕(1)ウ 送り仮名の正しい使い方 |
| 対義語・類義語 | 〔知識及び技能〕(1)オ 語句の量を増し、語感を磨き語彙を豊かにすること |
| 同音・同訓異字 | 〔知識及び技能〕(1)ウ 漢字の同音・同訓異字の理解 |
| 誤字訂正 | 〔知識及び技能〕(1)ウ 漢字を文や文章の中で正しく使う能力 |
| 四字熟語 | 〔知識及び技能〕(1)オ 慣用句・四字熟語など語彙の拡充 |
| 熟語の構成 | 〔知識及び技能〕(1)ウ・オ 熟語の構成についての理解 |

## 4. 年間指導計画（めやす）

3学期制・年間約35週を想定した配当例です。**あくまで練習の目安であり、
実際は各学級の国語科の進度（新出漢字の学習順）に合わせて前後させて
ください。** 6年生の新出配当漢字を無理なく既習化しつつ、後半に総合的
な出題形式（熟語の構成・誤字訂正など、既習知識を組み合わせて解く形式）
を配置しています。

| 学期 | 時期 | 週数目安 | 主な出題形式 |
|---|---|---|---|
| 1学期 | 4月〜7月 | 約14週 | 読み（3週）／書き取り（3週）／部首・部首名（3週）／画数（3週）／送り仮名（2週） |
| 2学期 | 9月〜12月 | 約13週 | 対義語・類義語（3週）／同音・同訓異字（3週）／四字熟語（3週）／誤字訂正（2週）／既習形式の復習（2週） |
| 3学期 | 1月〜3月 | 約8週 | 熟語の構成（2週）／「まとめてれんしゅう（ランダム10問）」で総復習（4週）／検定前の弱点補強（2週） |

- 各週は必修の授業時数ではなく、朝学習・家庭学習・すきま時間などでの
  自習ペースの目安です。
- 新出の小学6年配当漢字を扱う「読み」「書き取り」は、学級の漢字学習
  の進度（教科書の新出漢字の登場順）を追い越さないよう、教員が随時
  出題形式を選んで割り当てることを前提にしています。
- 3学期はすべての形式を横断する「まとめてれんしゅう」中心の総復習期
  とし、必要に応じて弱点の出題形式を個別に割り当ててください。

## 5. 使い方

1. **成績簿（hub/gradebook）から学級にモジュールを割り当てます。**
   担当学級を選び、「漢字検定 5級」を割り当てると、その学級の児童が
   ハブからアクセスできるようになります。
2. **児童は自分のペースで練習します。** 出題形式を選んで練習する
   ことも、「まとめてれんしゅう（10問）」でランダムに総合練習する
   こともできます。問題は毎回自動生成されるため、同じ問題の反復に
   よる「答えの丸暗記」が起きにくい設計です。

**運用のヒント**

- 学級の漢字学習の進度に合わせ、その週に習った漢字を含む出題形式
  （読み・書き取りなど）を中心に割り当てると、授業内容の定着に
  つながりやすくなります。
- 誤答が多い出題形式は成績簿の分析結果で確認できるので、朝学習や
  家庭学習で該当形式を重点的に割り当てる、といった個別対応にも
  活用できます。
