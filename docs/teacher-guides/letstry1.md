# 外国語活動 3年 — 先生向けガイド (module key: letstry1)

## 1. 概要

「Let's Try 1 練習」は、小学3年生の外国語活動で扱う単語・表現を、児童が自分のペースで
繰り返し練習できる学習コンパニオンです。Unit 1〜9の全9単元をカバーし、各単元につき
「単語クイズ（選択式）」「フラッシュカード」「マッチング（英語⇔日本語）」
「文を作ろう（語句の並べ替え）」の4種類の練習アクティビティを提供します。

**位置づけ：本モジュールは授業の補助・定着用の自主練習ツールであり、外国語活動の
授業そのものを代替するものではありません。** 実際の音声によるやり取り・チャンツ・
歌・ALTとのコミュニケーション活動は教室での指導が中心であり、本モジュールはその後の
語彙・表現の定着確認や家庭学習・すきま時間の復習として活用することを想定しています。

## 2. 単元一覧

| Unit | タイトル（英） | 内容（和文） | 語彙数（目安） |
|---|---|---|---|
| 1 | Hello! | 世界のいろいろなことばであいさつをしよう | 11語句 |
| 2 | How are you? | ごきげんいかが？気持ちを伝えよう | 11語句 |
| 3 | How many? | 数を数えよう！1から20までの数の言い方 | 11語句 |
| 4 | I like blue. | 好きな色やものを伝えよう | 11語句 |
| 5 | What do you like? | 好きなもの（色・食べ物・スポーツ）を伝えよう | 11語句 |
| 6 | ALPHABET | アルファベットの文字を読んだり書いたりしよう | 11語句 |
| 7 | This is for you. | 形や色を使ってほしいものを伝えよう | 11語句 |
| 8 | What's this? | ヒントを聞いて何かを当てよう | 11語句 |
| 9 | Who are you? | なりきりゲームをしよう！動物やものになりきる | 11語句 |

各単元とも「単語クイズ」「フラッシュカード」「マッチング」に加え、
4文前後の短い例文を並べ替える「文を作ろう」アクティビティを備えています。

## 3. 学習指導要領との対応（めやす）

> **※本節はめやす（要確認）です。** 各単元と学習指導要領上の「内容」「目標」との
> 対応づけは、モジュール内のテーマ・表現から作成した草案であり、正式な教育課程上の
> 位置づけとしては教育課程・外国語活動担当の専門家による確認を経ていません。
> 校内での活用にあたっては、最終的な整合性の確認をお願いします。

小学校学習指導要領（外国語活動・第3学年）の「知識及び技能」「思考力・判断力・
表現力等」の内容のうち、本モジュールの各単元が主に対応すると考えられる領域は
以下の通りです。

| Unit | 対応する学習指導要領上の主な内容（めやす） |
|---|---|
| 1. Hello! | 世界の様々な言語での挨拶を通した、言語や文化の多様性への気づき／初歩的な挨拶・自己紹介表現 |
| 2. How are you? | 気持ちや状態を表す表現を用いた簡単なやり取り |
| 3. How many? | 身の回りの数（1〜20）の言い方、数を尋ねる表現 |
| 4. I like blue. | 自分の好き嫌いを伝える表現、身の回りの色 |
| 5. What do you like? | 相手に好きなものを尋ねる表現、スポーツ・食べ物などの語彙 |
| 6. ALPHABET | アルファベットの文字（活字体）の読み方への慣れ親しみ |
| 7. This is for you. | ほしいものを伝える表現、形と色を組み合わせた言い方、やり取りを通じた相手への配慮 |
| 8. What's this? | ヒントを聞いて答えを推測するクイズ形式のやり取り、身の回りの生き物・自然の語彙 |
| 9. Who are you? | 動物になりきって自分について伝え合う表現、簡単な疑問文とその応答 |

## 4. 年間指導計画（めやす）

日本の公立小学校の標準的な年間指導時数・3学期制（1学期：4〜7月、2学期：9〜12月、
3学期：1〜3月、年間約35週）を前提とした、本モジュールの週配分のめやすです。
**実際の配当は、担当学級の外国語活動の授業進度に本モジュールの練習を合わせる形で
調整してください。**

| 学期 | 週 | 対応Unit | 備考 |
|---|---|---|---|
| 1学期（4〜7月・約15週） | 1〜4週 | Unit 1: Hello! | 年度はじめの挨拶・自己紹介 |
| | 5〜8週 | Unit 2: How are you? | 気持ちの表現 |
| | 9〜12週 | Unit 3: How many? | 数の言い方 |
| | 13〜15週 | Unit 4: I like blue.（前半） | 色・好みの表現 |
| 2学期（9〜12月・約14週） | 1〜3週 | Unit 4: I like blue.（続き） | 1学期からの継続 |
| | 4〜7週 | Unit 5: What do you like? | 好みを尋ねる表現 |
| | 8〜11週 | Unit 6: ALPHABET | 文字への慣れ親しみ |
| | 12〜14週 | Unit 7: This is for you.（前半） | 形・色・やり取り |
| 3学期（1〜3月・約6週） | 1〜2週 | Unit 7: This is for you.（続き） | 2学期からの継続 |
| | 3〜4週 | Unit 8: What's this? | クイズ形式のやり取り |
| | 5〜6週 | Unit 9: Who are you? | まとめ・なりきりゲーム |

※ 1〜9のUnit順は教科書の一般的な指導順に沿った並びを想定していますが、
実際に使用する教材・学級の年間指導計画とUnitの実施順が異なる場合は、
モジュール側のUnitタブ（U1〜U9）を自由な順番で選んで練習させることができます。

## 5. 使い方

1. **成績簿（hub/gradebook）からクラスに割り当てる。** 担当学級を選び、
   「letstry1」を課題として割り当てると、児童が各自の端末から自主練習できるように
   なります。
2. **児童は自分のペースで練習します。** 単元（Unit）を選んで、単語クイズ・
   フラッシュカード・マッチング・文を作ろう、の中から取り組みたいアクティビティを
   選べます。強制的な順番はないため、授業で扱ったUnitに絞って練習させたい場合は
   その旨を児童に伝えてください。

**運用のポイント**
- 授業で新しいUnitを導入した直後の1〜2週間は、そのUnitに絞って練習するよう
  声かけすると定着しやすくなります。
- 「マッチング」「文を作ろう」は語彙だけでなく表現のまとまりを扱うため、
  単語クイズ・フラッシュカードで語彙にある程度慣れてから取り組ませると効果的です。
