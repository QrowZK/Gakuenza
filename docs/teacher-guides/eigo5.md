# 外国語 5年 — 先生向けガイド (module key: eigo5)

## 1. 概要

`eigo5` は、小学5年生の外国語（英語）学習を対象とした、語い・文型の定着を
目的とする学習モジュールです。単語の意味を選ぶ問題と、英文の空らんに
正しい語句を入れる文型問題を中心に、8つの単元・全8ユニットで構成されて
います。

**本モジュールは授業を代替するものではなく、授業で扱った内容の定着・
補助練習を目的とした自学自習用の教材です。** 教室での対話・発音・
コミュニケーション活動と組み合わせ、家庭学習や授業内のすきま時間など
での反復練習用としてご活用ください。

## 2. 単元一覧

| No. | 単元名（このモジュールでの呼び方） | 内容の例 | 単語数（目安） | 文型問題数（目安） |
|---|---|---|---|---|
| u01 | 自己しょうかい・アルファベット | 名前・好きなもの・あいさつ表現 | 12語 | 5問 |
| u02 | たんじょう日と月 | 月の名前・たんじょう日の言い方 | 15語 | 5問 |
| u03 | 教科と時間わり | 教科名・曜日 | 16語 | 5問 |
| u04 | 一日の生活と時こく | 一日の行動・時こく表現 | 15語 | 5問 |
| u05 | できること (can) | can を使った「できること」表現 | 12語 | 5問 |
| u06 | 道あんないと場所 | 建物・場所・道あんない表現 | 15語 | 5問 |
| u07 | レストランとねだん | 食べ物・ねだんのたずね方 | 14語 | 5問 |
| u08 | あの人はだれ？ | 人がらを表す形容詞・職業 | 14語 | 5問 |

※単語数・問題数は現時点のコンテンツ量の目安です。今後の更新で追加される
場合があります。

## 3. 学習指導要領との対応（めやす）

> **本セクションは「めやす（要確認）」です。** 各単元と学習指導要領上の
> 内容・目標との対応は、教材開発時点での参考的な整理であり、教育課程の
> 専門家による確認前の草案です。実際のご指導・年間指導計画への正式な
> 反映にあたっては、必ず学校・教育委員会での確認をお願いします。

外国語科（小学5・6年）の学習指導要領では、聞くこと・話すこと（やり取り・
発表）・読むこと・書くことの4技能5領域と、以下のような言語材料・言語活動
の内容が示されています。`eigo5` の各単元は、次のように対応するものと
考えられます。

| 単元 | 対応する学習指導要領の内容・目標（めやす） |
|---|---|
| u01 自己しょうかい・アルファベット | 自分の名前・好みなどについて、簡単な語句や表現を用いて紹介し合う活動／アルファベットの読み方に慣れ親しむこと |
| u02 たんじょう日と月 | 月日や誕生日について、聞いたり尋ねたり答えたりする活動 |
| u03 教科と時間わり | 学校生活（教科・時間割）について、好みや事実を伝え合う活動 |
| u04 一日の生活と時こく | 日課や時刻について、簡単な語句や表現を用いて伝え合う活動 |
| u05 できること (can) | 自分や身近な人ができることについて、can を用いて伝え合う活動 |
| u06 道あんないと場所 | 道案内や場所について、簡単な語句や表現を用いて伝え合う活動 |
| u07 レストランとねだん | 買い物・飲食店での注文やねだんについて、伝え合う活動 |
| u08 あの人はだれ？ | 身近な人物の人がらや特徴について、形容詞を用いて紹介し合う活動 |

## 4. 年間指導計画（めやす）

日本の公立小学校の標準的な年間スケジュール（3学期制・年間約35週）を
前提とした、活用ペースの一例です。**実際の授業の進度に合わせて調整して
ください。** 本モジュールは自学自習用のため、クラスの教科書の進度に
応じて、担当単元の練習を割り当てる運用を想定しています。

| 学期 | 期間（目安） | 割り当てる単元 | 週数（目安） |
|---|---|---|---|
| 1学期 | 4月〜7月 | u01 自己しょうかい・アルファベット | 3週 |
| 1学期 | 4月〜7月 | u02 たんじょう日と月 | 3週 |
| 1学期 | 4月〜7月 | u03 教科と時間わり | 3週 |
| 1学期 | 4月〜7月 | u04 一日の生活と時こく（前半） | 3週 |
| 2学期 | 9月〜12月 | u04 一日の生活と時こく（後半・復習） | 2週 |
| 2学期 | 9月〜12月 | u05 できること (can) | 4週 |
| 2学期 | 9月〜12月 | u06 道あんないと場所 | 4週 |
| 3学期 | 1月〜3月 | u07 レストランとねだん | 4週 |
| 3学期 | 1月〜3月 | u08 あの人はだれ？ | 4週 |
| 3学期 | 1月〜3月 | 全単元 総復習 | 5週 |

※上記は一例であり、教科書会社・学校ごとの年間指導計画とは前後関係が
異なる場合があります。本モジュールの練習は、あくまで授業で扱った単元
に「あとから」合わせて割り当てる運用を想定しています。

## 5. 使い方

- クラスへの割り当ては、成績簿（gradebook）の「モジュール割り当て」
  画面から `eigo5` を選び、対象クラスと（必要に応じて）単元
  （focus_units）を指定して行います。単元を指定しない場合は、全8単元
  が対象になります。
- 児童は割り当てられた範囲内で、各自のペースで練習を行います。教員は
  成績簿から取り組み状況・正答状況を確認できます。
- **実践のポイント①**：授業でその単元を扱った直後の1〜2週間に絞って
  割り当てると、習ったばかりの語い・表現の定着に効果的です。
- **実践のポイント②**：学期末や単元のまとまりごとに、それまでの単元を
  まとめて再割り当てすると、総復習・忘却防止に活用できます。
