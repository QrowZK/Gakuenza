# 社会 4年 — 先生向けガイド（module key: shakai4）

## 1. 概要

本モジュールは、小学4年生「社会」の学習内容（石川県・羽咋市を題材とした
郷土学習）を、生徒が自分のペースで反復練習できる形にしたデジタル教材です。
各単元は「レッスン（読み物）」→「ドリル（一問一答・並べかえ形式の問題）」
→「ふりかえり」という流れで構成されており、地図や模式図（水の流れなど）
を使った視覚的な解説も含みます。内容はすべてオリジナル教材で、教科書本文
そのものではなく、独立して確認できる地理・歴史的事実にもとづいて作成され
ています。

本モジュールは、**授業の補助・定着用の補助教材**です。教室での説明・体験
活動・地図やハザードマップの読み取り指導など、社会科の中心となる学習を
置き換えるものではなく、既習内容の反復・定着や、家庭学習・すきま時間での
自主練習を想定しています。

## 2. 単元一覧

`units.js` に登録されている単元は以下の5つです（各単元は複数の「セクション」
に分かれ、セクションごとにレッスンとドリル問題があります）。

| # | 単元キー | 単元名 | セクション数（目安） |
|---|---|---|---|
| 1 | `u1_prefecture` | わたしたちの県 石川県 | 4（県のすがた／地形／羽咋市／産業と交通） |
| 2 | `u2_water_waste` | 健康なくらしを支える仕組み（水・ごみ） | 3（水はどこから／水を大切に・水源を守る／ごみのしょりと利用） |
| 3 | `u3_disaster_prep` | 自然災害からくらしを守る | 3（令和6年能登半島地震／地震・津波にそなえる／くらしを守る人々） |
| 4 | `u4_heritage_and_pioneers` | 県内の伝統・文化と先人の働き | 2（残したいもの伝えたいもの／先人の働き：板屋兵四郎と辰巳用水） |
| 5 | `u5_featured_areas` | 県内の特色ある地域 | 3（輪島市／金沢市／加賀市） |

合計 5単元・15セクション。各セクションに数問〜数問程度のドリル問題（一問一答・
並べかえ）が用意されています。

## 3. 学習指導要領との対応（めやす）

> **※ めやす（要確認）**：以下は本モジュールの単元と、小学校学習指導要領
> 社会科 第4学年の内容項目との対応関係を作成側で整理した草案です。実際の
> 学年内配当・文言の正確な対応については、教育課程を管理する立場（教務主任・
> 社会科主任等）でのご確認をお願いします。

| モジュール単元 | 対応する学習指導要領 内容（社会 第4学年） |
|---|---|
| 1. わたしたちの県 石川県 | 内容(1)「都道府県の様子」— 自都道府県の位置、地形や産業の概要、県内の地域構成などを捉える学習 |
| 2. 健康なくらしを支える仕組み | 内容(2)「人々の健康や生活環境を支える事業」— 飲料水、電気、ガス、廃棄物（ごみ）の処理などの中から選択して扱う事業について、その仕組みや働きを捉える学習 |
| 3. 自然災害からくらしを守る | 内容(3)「自然災害から人々を守る活動」— 地域や自治体・関係機関が行う防災の取組、地域の人々の協力（自助・共助・公助）などを捉える学習 |
| 4. 県内の伝統・文化と先人の働き | 内容(4)「県内の伝統や文化、先人の働き」— 地域に受け継がれる文化財や伝統行事、地域の発展に尽くした先人の具体的な働きを捉える学習 |
| 5. 県内の特色ある地域 | 内容(5)「県内の特色ある地域の様子」— 伝統的な産業、国際交流に取り組む地域など、特色ある地域の人々の生活を捉える学習 |

## 4. 年間指導計画（めやす）

日本の標準的な年間指導週数（約35週、3学期制）を前提とした配当の一例です。
実際の進度は、教科書会社・地域教材（副読本）の配列や学級の学習進度に
合わせて調整してください。本モジュールは**授業の進度に合わせて追いかける
形での復習・定着用**の利用を想定しています（先取り学習として使う場合は
順序を入れ替えてもかまいません）。

| 学期 | 期間 | 配当単元 | 目安週数 |
|---|---|---|---|
| 1学期 | 4月〜7月（約15週） | 1. わたしたちの県 石川県 | 5週 |
| | | 2. 健康なくらしを支える仕組み（水・ごみ） | 5週 |
| | | （導入・まとめ・学級行事等の予備） | 5週 |
| 2学期 | 9月〜12月（約14週） | 3. 自然災害からくらしを守る | 5週 |
| | | 4. 県内の伝統・文化と先人の働き | 5週 |
| | | （予備・行事等） | 4週 |
| 3学期 | 1月〜3月（約6週） | 5. 県内の特色ある地域 | 4週 |
| | | 総復習・全単元ふりかえり | 2週 |

※ 上記はあくまで目安であり、実際の学級の学習進度（教科書会社の年間指導
計画、行事や地域学習との兼ね合い等）に応じて前後します。教室での該当単元
の学習が終わった直後にモジュールで練習させると定着効果が高まります。

## 5. 使い方

- **配信**：ハブの成績簿（`hub/gradebook/assign.html`）から、担当学級に対して
  `shakai4` モジュールを割り当てます。特定の単元だけを練習させたい場合は、
  同画面の単元しぼり込み（focus_units）で対象単元にチェックを入れて配信
  できます。
- **進め方**：生徒はレッスン（読み物）→ドリル（一問一答・並べかえ）→
  ふりかえりの順に、各自のペースで進めます。教員側の操作は不要です。

**運用のヒント**
1. 教室で該当単元の授業が終わった週に配信すると、既習事項の定着確認として
   効果的です（先取り配信よりも復習配信を推奨）。
2. 単元3（自然災害）は、実際に石川県で発生した地震を扱う内容のため、防災
   学習や避難訓練の時期に合わせて配信すると、学習の実感が伴いやすくなります。
