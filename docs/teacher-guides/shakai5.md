# 社会 5年 — 先生向けガイド (module key: shakai5)

## 1. 概要

本モジュールは、小学5年生の社会科（国土・産業・環境）の学習内容を、
一問一答形式の練習問題とミニ解説（ふりがな付き用語・図解）で定着させる
教材です。世界の大陸・海洋、日本の国土と地形・気候、農業（米づくり・
水産業）、工業生産（自動車工業など）、情報化社会、自然災害・森林・
環境保全までを扱います。

児童は自分のペースでオリジナルの練習問題に取り組み、正誤に応じた
解説を読みながら知識を確認・定着させます。図（世界地図・日本列島の
四端・米づくりの流れ・自動車ができるまで・季節風など）はすべて本モジュール
オリジナルの模式図で、教科書の図版を複製したものではありません。

**本モジュールはあくまで授業の補助・定着用の教材であり、教科書や
授業そのものに代わるものではありません。** 新しい単元の導入や、教科書・
資料集を用いた本格的な学習は通常どおり授業で行い、本モジュールは
「授業で学んだ内容の反復・確認」「家庭学習」「単元末の理解度チェック」
としてご活用ください。

## 2. 単元一覧

モジュール内は5つの大単元・17の小テーマ（セクション）、合計約190問の
練習問題で構成されています。

| 大単元キー | 大単元名 | 小テーマ（セクション） |
|---|---|---|
| `u1_national_land` | 1. わたしたちの国土 | 世界の中の日本／日本のはんい（国土）／日本の地形／日本の気候／あたたかい・寒い・低い・高い土地のくらし |
| `u2_food_production` | 2. わたしたちの生活と食料生産 | 米づくりのさかんな地域／水産業のさかんな地域／これからの食料生産 |
| `u3_industrial_production` | 3. わたしたちの生活と工業生産 | 自動車をつくる工業／工業のさかんな地域／これからの工業生産と貿易 |
| `u4_information_society` | 4. 情報化した社会と産業の発展 | 情報を伝える／情報を生かす産業／情報を正しく使う |
| `u5_environment` | 5. わたしたちの生活と環境 | 自然災害からくらしを守る／森林とわたしたちのくらし／環境を守る |

各セクションにはミニ解説（用語の意味・図解）と、それに対応する練習問題
（一問一答・順序整序問題など）が10〜12問程度収録されています。

## 3. 学習指導要領との対応（めやす）※要確認

以下は学習指導要領（社会 第5学年）の内容区分との対応の**たたき台**です。
**正式な整合性の確認は行っておらず、公開・案内前にカリキュラム担当者
による確認をお願いします。**

| モジュール単元 | 対応する学習指導要領の内容（社会 第5学年） |
|---|---|
| 1. わたしたちの国土 | 内容(1) 我が国の国土の様子と国民生活（国土の位置、地形や気候の概要、国土の自然環境と国民生活との関連） |
| 2. わたしたちの生活と食料生産 | 内容(2) 我が国の農業や水産業における食料生産（食料生産の概要、生産・流通の工夫、食料生産を支える人々の工夫と努力、これからの食料生産） |
| 3. わたしたちの生活と工業生産 | 内容(3) 我が国の工業生産（工業生産の概要、工業製品ができるまでの工程、工業生産を支える貿易や運輸、これからの工業生産） |
| 4. 情報化した社会と産業の発展 | 内容(4) 我が国の産業と情報との関わり（情報を集め発信する産業の様子、情報化が国民生活に及ぼす影響、情報の受け手・送り手としての責任） |
| 5. わたしたちの生活と環境 | 内容(5) 我が国の国土の自然環境と国民生活との関連（自然災害の防止、森林資源の働き、公害の防止と生活環境の改善） |

## 4. 年間指導計画（めやす）

日本の標準的な年間授業週数（約35週、3学期制）を前提とした、モジュール
活用のペース配分の一例です。**実際の教科書・学校の年間指導計画の進度に
合わせて調整してください。あくまで目安であり、クラスの授業進度に
本モジュールの練習を追随させることを想定しています。**

| 学期 | 時期 | 主な単元 | 目安週数 |
|---|---|---|---|
| 1学期 | 4月〜7月 | 単元1「わたしたちの国土」（5テーマ） | 約8週 |
| 1学期 | 4月〜7月 | 単元2「わたしたちの生活と食料生産」（3テーマ） | 約6週 |
| 2学期 | 9月〜12月 | 単元3「わたしたちの生活と工業生産」（3テーマ） | 約7週 |
| 2学期 | 9月〜12月 | 単元4「情報化した社会と産業の発展」（3テーマ） | 約5週 |
| 3学期 | 1月〜3月 | 単元5「わたしたちの生活と環境」（3テーマ） | 約6週 |
| 3学期 | 1月〜3月 | 既習単元の復習・総まとめ | 約3週 |

（合計 約35週。上記は単元数・分量にもとづく目安配分であり、学校ごとの
使用教科書・年間指導計画により前後します。）

## 5. 使い方

- クラスへの割り当ては、成績簿（hub/gradebook）の割り当て画面から
  行います。対象クラスを選び、`shakai5` を割り当てると、そのクラスの
  児童が自分のアカウントから練習を開始できます。単元をしぼり込みたい
  場合は、割り当て画面の単元ピッカーで対象単元（例：`u2_food_production`
  のみ）を指定できます。指定しない場合は全単元が対象になります。
- 児童はそれぞれ自分のペースで練習を進めます。全員が同時に同じ問題を
  解く形式ではないため、進度の異なる児童にも柔軟に対応できます。

**実践のヒント：**
- 単元を教えた直後に該当セクションだけを絞って割り当てると、
  「授業で学んだことをその日のうちに復習する」使い方がしやすくなります。
- 3学期のまとめの時期は、単元をしぼらず全範囲を割り当て、既習内容の
  総復習として活用すると効果的です。
