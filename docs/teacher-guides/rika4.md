# 理科 4年 — 先生向けガイド（module key: rika4）

## 1. 概要

本モジュール（`rika4`）は、小学校4年生の理科で学習する内容を、選択式・並べ替え式の一問一答形式で練習できる自学自習用の教材です。単元ごとに「豆知識（要点）」の確認と、複数の設問（選択問題・順序並べ替え問題、一部は自動生成問題）を通じて、知識の定着と学習内容の振り返りを行うことができます。

**本モジュールはあくまで授業の補助・定着用の教材であり、理科の実験・観察を中心とした通常の授業に代わるものではありません。** 実際の観察・実験、考察の指導は教室での授業を通じて行い、本モジュールはその後の知識確認・復習・宿題等での活用を想定しています。

内容は東京書籍の教科書シリーズと同じ単元構成・進度感を参考にしつつ、設問・説明文はすべてオリジナルで作成しています（教科書の文章・図版・具体的な問題文をそのまま使用したものではありません）。

## 4. 年間指導計画（めやす）

日本の公立小学校の標準的な年間週数（約35週、3学期制）を前提とした、単元配置のめやすです。**実際の学習進度はクラス・学校の指導計画により前後するため、本モジュールでの練習は、成績簿（class_modules の focus_units 設定）を使って「今、授業で扱っている単元」に合わせて範囲を絞ることを想定しています。**

| 学期 | 期間 | 単元（学習順） | 週あたりの目安 |
|------|------|----------------|----------------|
| 1学期 | 4月〜7月 | u01 あたたかくなると → u02 動物のからだのつくりと運動 → u03 天気と気温 → u04 電流のはたらき → u05 雨水のゆくえと地面のようす | 各単元 2〜3週 |
| 2学期 | 9月〜12月 | u06 月や星の見え方 → u07 自然のなかの水のすがた → u08 とじこめた空気と水 → u09 物の体積と温度 | 各単元 3〜4週 |
| 3学期 | 1月〜3月 | u10 物のあたたまり方 → u11 水のすがたと温度 → u12 生き物の1年をふり返って（まとめ） | 各単元 2〜3週 |

※ 実際の教科書会社・年度によって単元の配当月は多少前後します。上表は目安として、授業の進度に合わせて適宜調整してください。

## 5. 使い方

- **成績簿（hub/gradebook）からクラスに割り当てます。** 「モジュール割当」画面で `rika4` を選び、必要に応じて `focus_units` で今扱っている単元にチェックを入れると、児童のメニュー上でその単元が優先的に表示されます（未設定の場合は全単元が対象になります）。
- 児童は自分のペースで、選んだ単元の設問に取り組みます。結果は成績簿の個票・設問別分析で確認できます。

**運用のヒント**
- 授業で単元を終えた直後〜数日以内に該当単元を割り当てると、知識の定着に効果的です。
- 単元末や学期末には `focus_units` の絞り込みを外し、既習範囲全体を対象にした復習として使うのもおすすめです。
