# 外国語 6年 — 先生向けガイド (module key: nh6)

## 1. 概要

`nh6`（New Horizons 6 練習）は、小学6年生の外国語（英語）教科書
「New Horizons 6」の内容に沿って作られた、自習用の練習モジュールです。
教科書の8つのユニットに対応する形で、文法練習・応答練習（会話のやり
とり）・書き練習（アルファベット・単語・文の書き取り）・単語練習の
4種類の活動を、選択式クイズと書き取り演習を中心に提供します。

**本モジュールは授業の補助・定着用の教材であり、授業そのものの代わ
りにはなりません。** 教室での聞く・話す・やりとりの指導を前提とし、
すでに学んだ表現や語彙を、児童が自分のペースで復習・定着させるため
のツールとしてご利用ください。

## 2. 単元一覧

`nh6` は教科書の8ユニットに対応する8つの単元で構成されています。
各単元には「文法練習」12問、「応答練習」10問、「書き練習」（文字・
単語・例文）7〜8問、および外部の単語アプリへのリンク（単語練習）が
あります。

## 4. 年間指導計画（めやす）

日本の年間約35週・3学期制（1学期：4〜7月、2学期：9〜12月、3学期：
1〜3月）を前提とした、教科書の指導順に沿う配分の一例です。
**実際の配分は、クラスの教科書の進度に合わせて調整してください
（本モジュールは授業の進度を追いかける形での利用を想定しています）。**

| 学期 | 単元 | めやす週数 |
|---|---|---|
| 1学期（4〜7月） | 1. This is me! | 3〜4週 |
| 1学期（4〜7月） | 2. My Daily Schedule | 3〜4週 |
| 1学期（4〜7月） | 3. My Weekend | 3週 |
| 2学期（9〜12月） | 4. Let's see the world. | 3〜4週 |
| 2学期（9〜12月） | 5. Where is it from? | 3週 |
| 2学期（9〜12月） | 6. Save the animals. | 3〜4週 |
| 3学期（1〜3月） | 7. My Best Memory | 3週 |
| 3学期（1〜3月） | 8. My Future, My Dream | 3〜4週 |

※ 各単元とも、授業で新出表現を学んだ後の1〜2週間は「文法練習」
「応答練習」を中心に、単元のまとめ時期には「書き練習」も合わせて
取り組ませると定着しやすくなります。

## 5. 使い方

1. **成績簿（がくえん座 Hub）から、担当クラスに `New Horizons 6 練習`
   を割り当ててください。** 割り当て後、児童は各自のアカウントで
   ログインし、自分のペースで単元・活動を選んで練習できます。
2. 児童の解答結果は自動的に成績簿へ記録されるため、単元ごとの正答率
   をもとに、つまずいている表現を把握し、次の授業の復習ポイントを
   決める材料として活用できます。

**運用のヒント**
- 教科書で新出表現を導入したその週のうちに「文法練習」「応答練習」
  を家庭学習や隙間時間の課題として割り当てると、聞く・話すの授業と
  記憶の定着がつながりやすくなります。
- 「書き練習」（アルファベット・単語・例文の書き取り）は自動採点さ
  れないため、進捗の目安として活用し、実際の書字の丁寧さは別途机間
  巡視等で確認することをおすすめします。
