# 国語 1年 — 先生向けガイド (module key: kokugo1)

## 1. 概要

`kokugo1` は、小学1年生の国語のうち「文字の読み書きの基礎」に特化した学習アプリです。ひらがな・カタカナの読み書き、助詞「は・を・へ」の使い分け、句読点・かぎかっこの使い方、そして学年配当漢字80字の読み・字形・画数までを、選択式のクイズ形式で反復練習できます。

本モジュールは**授業の補助・定着用の教材**です。ひらがな・カタカナの導入や漢字の書き順指導など、実際の指導は教室での対面授業が前提であり、`kokugo1` はその内容を教えるものではなく、すでに授業で扱った内容を児童が自分のペースで繰り返し練習し、定着させるためのものです。授業の代替ではない点にご留意ください。

- 対象学年：小学1年生
- 教科：国語（文字・言葉のきまり・漢字）
- 形式：4択クイズ（1回10問）、正誤フィードバックと見直し画面つき

## 2. 単元一覧

`kokugo1` には次の5つの単元（モード）があります。

| 単元キー | 単元名 | 内容 |
|---|---|---|
| `hiragana` | ひらがな | 五十音の読み、濁音・半濁音、促音（っ）・拗音（ゃゅょ）・長音を含む語の正しい書き方 |
| `katakana` | カタカナ | 五十音の書き（ひらがな→カタカナ変換）、外来語の表記（長音符「ー」を含む語） |
| `joshi` | は・を・へ | 助詞「は」「を」「へ」の使い分け（「わ」「お」「え」との書き分けを含む） |
| `kutouten` | 。、「」 | 句点（。）・読点（、）・かぎかっこ（「」）の正しい使い方 |
| `kanji` | かんじ（80字） | 学年配当漢字80字（MEXT学年別漢字配当表準拠）の読み方・漢字選び・画数の問題 |

いずれも単元選択画面から児童が自分でモードを選び、10問1セットで出題されます。かな系の3単元（ひらがな・カタカナ・助詞）と句読点単元は文型・語彙バンクからの出題、漢字単元は80字の配当漢字からの出題です。

## 3. 学習指導要領との対応（めやす）

> **※ここは仮の対応表（めやす）です。正確性については必ず国語科の専門家・カリキュラム担当者の確認を受けてください。**

小学校学習指導要領 国語 第1学年及び第2学年の「知識及び技能」内容(1)に基づく、大まかな対応の目安です。

| 単元キー | 対応する学習指導要領上の内容（1年生相当） |
|---|---|
| `hiragana` | (1)イ 音節と文字との関係、拗音・促音・長音などの表記、ひらがなを読み、書くこと |
| `katakana` | (1)イ 片仮名を読み、書くこと（主に外来語・擬音語等の表記に用いる長音符号を含む） |
| `joshi` | (1)ウ 助詞の「は」「へ」「を」の使い方（文の中における意味の通じ方） |
| `kutouten` | (1)ウ 句読点の打ち方、かぎ（`「」`）の使い方を理解し、文や文章の中で使うこと |
| `kanji` | (1)エ・別表 学年別漢字配当表に示された第1学年の80字を読み、漸次書くこと |

## 4. 年間指導計画（めやす）

> 3学期制・年間約35週を想定した、単元配当の目安です。実際の指導進度に合わせて調整してください。国語の教科書は「ひらがな→カタカナ→助詞・句読点→漢字」の順に進むのが一般的なため、この順序に沿って配置していますが、**あくまで練習アプリの割り当ての目安であり、クラスの授業進度に合わせて `focus_units`（成績簿の課題範囲設定）で調整することを前提とします。**

| 学期 | 期間 | 週数（目安） | 重点単元 | 補足 |
|---|---|---|---|---|
| 1学期 | 4月〜7月 | 約15週 | `hiragana`（第1〜8週）→ `katakana`（第9〜13週）→ `joshi`（第14〜15週） | ひらがな導入直後から並行して練習を開始できる。カタカナは夏休み前までに一巡させる想定。 |
| 2学期 | 9月〜12月 | 約14週 | `joshi` 継続（第1〜2週）→ `kutouten`（第3〜6週）→ `kanji` 導入（第7〜14週） | 助詞・句読点の定着を秋のうちに固め、漢字学習が本格化する時期に合わせて `kanji` 単元を開放する。 |
| 3学期 | 1月〜3月 | 約8週 | `kanji` 継続・総復習（全8週） | 80字の配当漢字を計画的に振り返り、全単元の総復習期間として活用する。 |

## 5. 使い方

1. 成績簿（`hub/gradebook`）から対象クラスに `kokugo1` を割り当てます。教材の進度に応じて「今週の重点単元（focus_units）」を設定すると、単元選択画面で該当モードに「今週」バッジが表示され、児童が今取り組むべき単元が視覚的にわかりやすくなります。
2. 児童は自分のアカウントでログインし、単元を選んで自分のペースで10問ずつ練習します。結果は自動的に成績簿へ記録されます。

**運用のヒント**

- ひらがな・カタカナ単元は繰り返し取り組んでも出題がランダムに変化するため、朝学習や隙間時間の反復練習に向いています。
- 漢字単元（`kanji`）は80字全体から出題されるため、配当漢字を授業で一通り導入し終えてから `focus_units` で開放すると、未習の字が出題される混乱を避けられます。
