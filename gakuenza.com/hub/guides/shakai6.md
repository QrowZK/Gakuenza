# 社会 6年 — 先生向けガイド（module key: shakai6）

## 1. 概要

本モジュールは、小学6年生「社会」の学習内容（政治のしくみ・日本の歴史・
国際社会とのつながり）を、生徒が自分のペースで反復練習できる形にした
デジタル教材です。各単元・各セクションは「レッスン（読み物）」→「ドリル
（一問一答・並べかえ形式の問題）」という流れで構成されており、日本国憲法
の三原則や三権分立などは、オリジナルの模式図（SVG）も使って視覚的に解説
しています。内容はすべてオリジナル教材で、教科書本文そのものではなく、
独立して確認できる一般的な事実（年代・人物・制度・因果関係など）にもとづ
いて作成されています。

本モジュールは、**授業の補助・定着用の補助教材**です。教室での説明・資料
活用・話し合い活動など、社会科の中心となる学習を置き換えるものではなく、
既習内容の反復・定着や、家庭学習・すきま時間での自主練習を想定しています。

> 学年内配当・文言の正確な対応については、教育課程を管理する立場（教務主任・

## 4. 年間指導計画（めやす）

日本の標準的な年間指導週数（約35週、3学期制）を前提とした配当の一例です。
実際の進度は、教科書会社の配列や学級の学習進度に合わせて調整してくだ
さい。本モジュールは**授業の進度に合わせて追いかける形での復習・定着用**
の利用を想定しています（歴史単元は必ず時代順に進める前提です）。

| 学期 | 期間 | 配当単元 | 目安週数 |
|---|---|---|---|
| 1学期 | 4月〜7月（約15週） | 1. わたしたちの生活と政治（4セクション） | 4週 |
| | | 2-①〜⑤ 縄文〜室町文化 | 9週 |
| | | （導入・まとめ・学級行事等の予備） | 2週 |
| 2学期 | 9月〜12月（約14週） | 2-⑥〜⑫ 戦国〜新しい日本・平和な日本へ | 11週 |
| | | （予備・行事等） | 3週 |
| 3学期 | 1月〜3月（約6週） | 3. 世界の中の日本（2セクション） | 3週 |
| | | 総復習・全単元ふりかえり | 3週 |

※ 上記はあくまで目安であり、実際の学級の学習進度（教科書会社の年間指導
計画、行事や修学旅行等との兼ね合い等）に応じて前後します。特に歴史単元
（単元2）は分量が多いため、教室の授業進度に対して配信が遅れないよう、
学期の早い段階からの並行配信をおすすめします。

## 5. 使い方

- **配信**：ハブの成績簿（`hub/gradebook/assign.html`）から、担当学級に対して
  `shakai6` モジュールを割り当てます。特定の単元・時代だけを練習させたい
  場合は、同画面の単元しぼり込み（focus_units）で対象セクションにチェック
  を入れて配信できます。
- **進め方**：生徒はレッスン（読み物）→ドリル（一問一答・並べかえ）の順に、
  各自のペースで進めます。教員側の操作は不要です。

**運用のヒント**
1. 単元2（歴史）は12時代分とボリュームが大きいため、教室の単元進度に
   合わせて2〜3時代ずつ小分けに配信すると、生徒の負担が偏らず定着効果も
   高まります。
2. 教室で該当セクションの授業が終わった週に配信すると、既習事項の復習・
   定着確認として効果的です（先取り配信よりも復習配信を推奨）。
