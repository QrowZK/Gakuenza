# 国語 2年 — 先生向けガイド (module key: kokugo2)

## 1. 概要

`kokugo2` は、小学校2年生の国語（漢字・言葉のきまり）を対象とした
自習型の練習モジュールです。児童は「かん字れんしゅう」または「ことば・
文のきまり」のどちらかを選び、4択形式の一問一答を10問1セットで解いて
いきます。出題は固定の問題バンクではなく、そのつどランダムに生成され
るため、同じ児童が繰り返し取り組んでも問題文・選択肢の組み合わせが毎
回変わります。

本モジュールは、あくまで**授業の補助・定着用の教材**です。読解・作文・
音読・話し合い活動など、教科書に沿った本来の国語指導を代替するもので
はなく、漢字の読み書きや言葉のきまりの「知識・技能」面を、授業外や隙
間時間に児童が自分のペースで反復練習するためのツールとしてご利用くだ
さい。

## 2. 単元一覧

`modules/kokugo2/units.js` に登録されている、本モジュールの実際の単
元（focus_units として成績簿から絞り込み指定も可能）は以下の6つです。

| 単元キー | 単元名 | 内容 |
|---|---|---|
| `kanji` | かん字れんしゅう | 2年生配当漢字 全160字（MEXT学年別漢字配当表準拠）。読み方・かん字えらび・画数を出題 |
| `katakana` | ことば：カタカナ | ひらがな⇔カタカナの字形対応、外来語・擬音語をカタカナで書く運用問題 |
| `kanazukai` | ことば：かなづかい | 助詞「は・を・へ」の正しい表記、「ぢ／じ」「づ／ず」、のばす音（長音）など特殊な仮名づかい |
| `shugo_jutsugo` | ことば：主語と述語 | 短文から「主語（何が・だれが）」と「述語（どうする）」を見つける |
| `nakama` | ことば：なかま・反対の言葉 | 反対語（対義語）、なかまの言葉（カテゴリー分類）、仲間はずれ探し |
| `kutouten` | ことば：丸・点・かぎ | 句点「。」・読点「、」・かぎかっこ「」・疑問符「？」の名称と使い方 |

かん字れんしゅう（`kanji`）はモジュール内で1つの大きな単元として扱わ
れ、160字全体からランダムに出題されます（学期ごとの配当字での絞り込
みは行っていません）。「ことば・文のきまり」側は上記5つの単元（`katakana`
〜`kutouten`）から児童が選んで練習します。

## 3. 学習指導要領との対応（めやす）※要確認

**この対応表は暫定案（めやす）です。国語科の指導要領・教科書会社別の
年間指導計画との整合性については、教科専門の担当者による確認をお願い
します。**

| 単元 | 学習指導要領上の位置づけ（小学校国語 第1学年及び第2学年） |
|---|---|
| `kanji` | 〔知識及び技能〕(1)エ「第2学年までに配当されている漢字を読むこと。また、第2学年に配当されている漢字を漸次書き、文や文章の中で使うこと」 |
| `katakana` | 〔知識及び技能〕(1)ウ「片仮名を読み、書くとともに、片仮名で書く語の種類を知り、文や文章の中で使うこと」 |
| `kanazukai` | 〔知識及び技能〕(1)ウ「助詞の『は』『へ』及び『を』の使い方、拗音、促音、長音などの表記」に関する事項 |
| `shugo_jutsugo` | 〔知識及び技能〕(1)カ「主語と述語との関係に気付くこと」 |
| `nakama` | 〔知識及び技能〕(1)オ「身近なことを表す語句の量を増し、話や文章の中で使うとともに、言葉には意味による語句のまとまりがあることに気付き、語彙を豊かにすること」（対義語・仲間の言葉の理解） |
| `kutouten` | 〔知識及び技能〕(1)ウ「句読点の打ち方、かぎ（`「」`）の使い方を理解して文や文章の中で使うこと」 |

## 4. 年間指導計画（めやす）

年間およそ35週（3学期制：1学期＝4〜7月、2学期＝9〜12月、3学期＝1〜3月）
を想定した、practice（練習）配分のめやすです。**授業での学習単元の
進度に合わせて活用することを前提とした目安であり、固定のカリキュラム
ではありません。** 実際の教科書配列・学級の進度に応じて前後させてくだ
さい。

| 学期 | 週 | 主な単元 | ねらい |
|---|---|---|---|
| 1学期 | 第1〜4週 | `kanazukai`（かなづかい） | 1年生の復習も兼ね、「は・を・へ」など基礎の仮名づかいを固める |
| 1学期 | 第5〜8週 | `katakana`（カタカナ） | 片仮名の字形と外来語表記に慣れる |
| 1学期 | 第9〜14週 | `kanji`（かん字れんしゅう） | 1学期配当漢字を中心に反復練習を開始（通年で継続） |
| 2学期 | 第15〜18週 | `shugo_jutsugo`（主語と述語） | 文の構造（主語・述語）の理解を進める |
| 2学期 | 第19〜24週 | `kanji`（かん字れんしゅう・継続） | 2学期配当漢字を含め、既習漢字も交えて反復 |
| 2学期 | 第25〜27週 | `nakama`（なかま・反対の言葉） | 語彙を広げ、言葉の関係性への気づきを深める |
| 3学期 | 第28〜31週 | `kutouten`（丸・点・かぎ） | 句読点・かぎかっこの使い方を仕上げる |
| 3学期 | 第32〜35週 | `kanji`＋総復習（全単元） | 学年末に向け、全単元を横断して定着を確認 |

`kanji`（漢字）は範囲が広い（160字）ため、上表のように特定の週に限定
せず、学期を通じて継続的に取り組ませることを推奨します。

## 5. 使い方

- **割り当て**：成績簿（`hub/gradebook/assign.html`）から担当クラスへ
  `kokugo2` を割り当てます。上記の単元キー（`kanji` / `katakana` /
  `kanazukai` / `shugo_jutsugo` / `nakama` / `kutouten`）を使って、そ
  の週に重点的に取り組ませたい単元を絞り込み指定（focus_units）できま
  す。指定した単元には画面上に「今週」バッジが表示されます。
- **取り組み方**：児童は自分のペースで、モード（かん字／ことば・文の
  きまり）を選んで10問1セットに取り組みます。1回あたり数分で完了する
  ため、朝学習や隙間時間、家庭学習にも向いています。
- **ペース配分のヒント**：
  1. 新しい単元（かなづかい・主語述語など）を授業で扱った直後の1〜2週
     は、その単元にfocus_unitsを絞ることで定着を後押しできます。
  2. 漢字は範囲が広いため、学期の早い段階から継続的に触れさせておく
     と、学年末の総復習が楽になります。
