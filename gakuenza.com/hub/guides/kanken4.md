# 漢字検定 4級 — 先生向けガイド (module key: kanken4)

## 1. 概要

`kanken4` は、日本漢字能力検定（漢検）4級の出題範囲に対応した漢字学習モ
ジュールです。4級の配当漢字313字（5級までの累計1026字に加えて新たに配
当される字。累計1339字＝中学校在学程度）と、4級の出題形式である読み・
書き取り・部首・送り仮名・対義語類義語・同音同訓異字・誤字訂正・四字熟
語・熟語の構成の9形式を、毎回自動生成される選択式問題で練習します。

出題文・熟語例・四字熟語などはすべて本アプリのために独自に書き起こした
オリジナル問題であり、協会の過去問題そのものは一切使用していません（出
題範囲・難易度のみ協会公表の審査基準に準拠）。

児童は主に高学年〜中学進学準備期を想定していますが、対象は漢検4級の学
習内容そのものであり、特定の学年の教科書単元と1対1には対応しません。
**本モジュールは授業の補助・定着用の自主学習教材であり、教室での漢字指
導・語彙指導に代わるものではありません。** 新出字・新出語の指導は授業で
行い、本モジュールはその後の反復練習・定着確認・検定対策として位置づけ
てください。

## 2. 単元一覧

`kanken4` は「単元」ではなく、漢検4級の**出題形式（カテゴリ）**を練習の
単位としています（`units.js` による単元ピッカーはありません＝授業の進度
に応じて特定の範囲だけを割り当てる機能は現時点でなく、全形式が常に選べ
ます）。児童は各カテゴリを個別に選ぶか、「まとめて練習」で全形式からラ
ンダムに10問を解きます。

| 出題形式 | 内容 | 問題データ規模（目安） |
|---|---|---|
| 読み | 熟語の読み方を選ぶ | 熟語56語 |
| 書き取り | 読みから正しい漢字表記を選ぶ | 熟語56語 |
| 部首・部首名 | 漢字の部首とその名称を選ぶ | 漢字76字分 |
| 送り仮名 | 正しい漢字＋送り仮名の形を選ぶ | 訓読み語22語 |
| 対義語・類義語 | 対になる語・似た意味の語を選ぶ | 語のペア19組 |
| 同音・同訓異字 | 文脈に合う同音（同訓）の漢字を選ぶ | 例文14件 |
| 誤字訂正 | 文中の誤って使われた同音字を正しく直す（4級から新規） | 例文12件 |
| 四字熟語 | 四字熟語の空欄に入る漢字を選ぶ | 熟語18語 |
| 熟語の構成 | 二字熟語の組み立て方を5択から見分ける | 熟語56語（読みと共用） |

土台となる配当漢字データは313字（`kanken4-data.js`）で、上記のうち
「読み」「書き取り」「熟語の構成」はこの313字の範囲内で作られた熟語56語
から、「部首・部首名」はこのうち76字分の部首データから出題されます。

## 3. 学習指導要領との対応（めやす・要確認）

> **この章は暫定的なめやすです。国語科カリキュラム担当者による確認前の
> 下書きであり、そのまま確定情報として保護者・教育委員会等へ提示しない
> でください。**

本モジュールは特定学年の学習指導要領「国語」単元にひもづく教材ではな
く、漢検4級の配当漢字（累計1339字＝小学校6年間の学習漢字1026字＋中学校
在学程度で新たに加わる313字）と、それに対応する学習指導要領上の「漢字
指導」「語彙指導」「言葉の特徴や使い方に関する事項」の内容を横断的に練
習するものと位置づけられます。

| 出題形式 | 対応する学習指導要領（国語）の内容・目標（めやす） |
|---|---|
| 読み・書き取り | 〔知識及び技能〕(1)ウ 学年別漢字配当表に基づく漢字の読み書き（4級は中学校段階の漢字を含むため、小学校配当分の定着＋発展的漢字の先取り練習として位置づけ） |
| 部首・部首名 | 〔知識及び技能〕(1)ウ 漢字の構成要素（部首）についての理解、第3〜4学年で扱う「へん・つくり」等の部首の考え方の発展 |
| 送り仮名 | 〔知識及び技能〕(1)ウ 送り仮名の付け方に関する事項 |
| 対義語・類義語 | 〔知識及び技能〕(1)オ 語彙（様子や心情を表す語句、意味による語句のまとまり）の理解と語彙を豊かにする指導事項 |
| 同音・同訓異字／誤字訂正 | 〔知識及び技能〕(1)ウ・オ 同音異字・同訓異字の使い分け、文脈に応じた正しい漢字選択（誤字訂正は漢検4級独自の出題形式で、指導要領上は語彙・表記の正確さに関する事項に相当） |
| 四字熟語 | 〔知識及び技能〕(1)オ 慣用句・故事成語等、伝統的な言語文化に関する語彙の理解 |
| 熟語の構成 | 〔知識及び技能〕(1)オ 熟語の構成の理解（第5〜6学年で扱う「熟語の成り立ち」の理解の発展） |

## 4. 年間指導計画（めやす）

日本の公立小学校の標準的な年間週数（約35週、3学期制）を前提に、9形式を
自然な指導順（漢検の公式カテゴリ順：読み→書き取り→部首→送り仮名→対義
語類義語→同音同訓異字→誤字訂正→四字熟語→熟語の構成）でならした配分案
です。**実際の週配分は、各クラスの国語の授業進度（新出漢字・語彙の学習
状況）に本モジュールでの練習が追随することを前提としています。**新出単
元の指導が先、本モジュールでの反復練習が後、という順序を崩さないでくだ
さい。

| 学期 | 期間目安 | 週数目安 | 重点カテゴリ |
|---|---|---|---|
| 1学期 | 4月〜7月 | 約14週 | 読み（3週）／書き取り（3週）／部首・部首名（3週）／送り仮名（3週）／導入としての「まとめて練習」（2週） |
| 2学期 | 9月〜12月 | 約13週 | 対義語・類義語（3週）／同音・同訓異字（3週）／誤字訂正（3週）／四字熟語（2週）／熟語の構成（2週） |
| 3学期 | 1月〜3月 | 約8週 | 全形式の総復習「まとめて練習」を中心に据え、誤答の多いカテゴリを個別強化（弱点別に4〜6週）／検定本番を想定した仕上げ演習（2週） |

## 5. 使い方

- 割り当ては成績簿（`hub/gradebook`）から、担当クラスに対して
  `kanken4` を登録してください。単元（focus_units）による範囲限定機能は
  本モジュールにはないため、割り当てると全出題形式が対象になります。
- 児童は選択画面で出題形式（または「まとめて練習」）を選び、自分のペー
  スで取り組みます。問題は毎回自動生成されるため、同じ問題の暗記による
  「答え合わせ」は起きにくい設計です。
- 実践のヒント：
  1. 週の前半は個別カテゴリ（例：今週は「送り仮名」だけ）で弱点を絞っ
     て練習させ、週の後半に「まとめて練習」で定着確認をする、という2段
     構えが効果的です。
  2. 「誤字訂正」「同音・同訓異字」はデータ規模が他より小さい（12〜14
     件）ため、短時間で一巡してしまいます。この2形式は反復練習よりも、
     授業内での用例確認とセットにして深める使い方が向いています。
