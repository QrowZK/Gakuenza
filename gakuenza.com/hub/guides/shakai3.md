# 社会 3年 — 先生向けガイド (module key: shakai3)

## 1. 概要

`shakai3` は、小学3年生の社会科（地域学習）を対象とした、Gakuenza上の
自習型ドリルモジュールです。方位・地図記号の読み取り、地域の生産・
販売の仕事、地域の安全を守る仕事、市のうつりかわりなど、3年社会の
主要テーマを一問一答形式・順序並べかえ形式の問題で反復練習できます。
図（オリジナルの町の地図・人口グラフ）を使った資料読み取り問題も
含まれます。

**本モジュールは授業の補助・定着用の教材であり、授業そのものの
代わりにはなりません。** まちたんけんや聞き取り調査、資料館見学と
いった体験的な学習活動、また教科書に基づく本時の指導は、これまで
どおり教室で行ってください。本モジュールは、そこで学んだ知識・
用語を児童が自分のペースで確認・定着させるためのものです。

## 2. 単元一覧

モジュールは大単元4つ・小単元（セクション）12から構成されています。

| 大単元 | 小単元（セクション） | 主な内容 |
|---|---|---|
| 1. わたしのまち みんなのまち | 1. 方位のきほん | 四方位・八方位、方位じしん |
| | 2. まちたんけんと絵地図 | 白地図・絵地図、まちの様子の違い |
| | 3. 地図記号 | 学校・郵便局・神社・寺・交番・消防署・市役所・田/畑/くだもの畑 等 |
| | 4. 市の様子 | 公共しせつ・税金、土地の高さと使われ方 |
| | 5. 地図を読み取ろう | オリジナルの町の地図を使った方位読み取り演習 |
| 2. はたらく人とわたしたちのくらし | 1. 農家の仕事 | 仕事ごよみ、ビニールハウス・たいひ、JA、出荷の流れ |
| | 2. 工場の仕事 | 原料〜製品、衛生管理、出荷の流れ |
| | 3. 店ではたらく人 | スーパーマーケットのくふう、産地、輸入 |
| 3. くらしを守る | 1. 火事からくらしを守る | 119番、通信指令室、消防署・消防団 |
| | 2. 事故や事件からくらしを守る | 110番、けいさつの仕事、こども110番の家 |
| 4. 市のうつりかわり | 1. 道具とくらしのうつりかわり | 昔の道具、時代の呼び方（明治〜令和） |
| | 2. 市の様子のうつりかわり | 年表、人口・交通・土地利用の変化 |
| | 3. 人口グラフを読み取ろう | オリジナルの人口グラフを使った資料読み取り演習 |

各セクションに数問〜10問程度の一問一答・順序並べかえ問題があり、
モジュール全体でおよそ70問の練習量があります。

なお `shakai3` には単元選択機能（`class_modules.focus_units` による
出題範囲の絞り込み）は現時点で用意されていません。成績簿から
割り当てると、児童は上記12セクションすべてに取り組むことができます。

## 3. 学習指導要領との対応（めやす）※要確認

以下は小学校学習指導要領・社会科（第3学年）の内容区分に沿った
対応関係の**めやす**です。**カリキュラム専門家による確認前の暫定的な
対応表であり、正式な整合性の保証ではありません。** 教育委員会・
校内研修等で使用する際は、必ず確認のうえご利用ください。

| モジュール内の大単元 | 対応する学習指導要領の内容区分（社会3年） |
|---|---|
| 1. わたしのまち みんなのまち | 内容(1) 身近な地域や市区町村の様子（方位・地図記号・土地利用・公共施設などを捉え、身近な地域や市の様子を大まかに理解する） |
| 2. はたらく人とわたしたちのくらし | 内容(2) 地域に見られる生産や販売の仕事（農家・工場・販売店の仕事の様子と、それが自分たちの生活を支えていることの理解） |
| 3. くらしを守る | 内容(3) 地域の安全を守る働き（消防署・警察署などの関係機関の働きと、地域の人々との協力） |
| 4. 市のうつりかわり | 内容(4) 市の様子の移り変わり（人々の生活の変化、道具の変化、交通・土地利用の変化などを時期による違いに着目して捉える） |

## 4. 年間指導計画（めやす）

日本の標準的な年間授業週数（約35週、3学期制）を前提とした配当の
めやすです。**あくまで目安であり、実際の教科書会社の年間指導計画や
学級の進度に合わせて調整してください。** モジュールの練習は、
授業の単元進行に合わせて割り当てることを想定しています。

| 学期 | 時期 | 割り当てる大単元 | 目安週数 |
|---|---|---|---|
| 1学期 | 4月〜7月 | 1. わたしのまち みんなのまち（方位・地図記号・市の様子） | 約6〜7週 |
| 1学期 | 4月〜7月 | 2. はたらく人とわたしたちのくらし（農家・工場の仕事） | 約4〜5週 |
| 2学期 | 9月〜12月 | 2. はたらく人とわたしたちのくらし（店ではたらく人・つづき） | 約2週 |
| 2学期 | 9月〜12月 | 3. くらしを守る（火事・事故事件からくらしを守る） | 約5〜6週 |
| 2学期 | 9月〜12月 | 4. 市のうつりかわり（前半：道具とくらしのうつりかわり） | 約3週 |
| 3学期 | 1月〜3月 | 4. 市のうつりかわり（後半：市の様子・人口グラフの読み取り） | 約4週 |

上記は各大単元の授業終了直後にモジュールで練習・定着を図ることを
想定した配当です。実際の授業進度が前後する場合は、モジュールの
割り当て時期もそれに合わせてずらしてください。

## 5. 使い方

- 成績簿（`hub/gradebook`）の割り当て画面から、担当クラスに
  `shakai3` を割り当ててください。単元の絞り込みはできないため、
  割り当てると児童は全12セクションに取り組めるようになります。
- 児童は各自のペースで、レッスンカード（解説）→ドリル（問題）
  →ふりかえりの順に進めます。1回で全セクションを終わらせる必要は
  なく、何度でもやり直せます。

**運用のヒント**
- 授業で扱った単元（大単元1〜4のいずれか）が終わったタイミングで
  割り当てると、その場で学んだ内容を児童が自分のペースで
  復習・定着できます。単元をまたいで一度に割り当てるより、
  進度に合わせて区切って使う方が定着しやすくなります。
- 地図記号や方位、119番・110番の仕組みなど、暗記要素の強い
  単元（1・3）は繰り返し取り組ませることで定着しやすく、
  グラフ・地図の読み取り単元（1-5・4-3）は授業内で一度解き方を
  確認したうえで宿題等に活用すると効果的です。
