# 算数 1年 — 先生向けガイド (module key: sansu1)

## 1. 概要

`sansu1` は、小学1年生の算数を対象にした自動生成ドリル形式の学習モジュールです。数と計算（10までのかず・20までのかず・100までのかず・くり上がり／くり下がりのある計算など）、時計、量の比較、図形といった1年生算数の主要単元をカバーしています。問題は毎回パラメータをランダムに変えて自動生成されるため、同じ児童が繰り返し取り組んでも同じ問題は出にくく、反復練習に向いています。

本モジュールは **授業の補助・定着用の教材** であり、教室での指導そのものを代替するものではありません。新しい概念の導入や、つまずきへの個別対応は教員による指導を前提とし、本モジュールはその後の習熟・定着（計算練習、既習事項の振り返り）を児童が自分のペースで行うためのツールとして位置づけてください。

## 3. 学習指導要領との対応（めやす）

**※ この節は めやす（要確認） です。実際の運用前に、教科教育の専門家による確認をお願いします。**

小学校学習指導要領 算数 第1学年「A 数と計算」「B 図形」「C 測定」の内容区分に沿った、大まかな対応関係の目安です。

| module単元 | 学習指導要領上の対応内容（めやす） |
|---|---|
| 10までのかず / なんばんめ | A(1) 数の意味と表し方（ものの個数を数え、数の大小・順序を理解する） |
| いくつといくつ | A(1) 数の合成・分解（10までの数の構成） |
| たしざん(1) / ひきざん(1) | A(2) 加法・減法の意味（1位数どうしの加減、繰り上がり・繰り下がりのないもの） |
| 20までのかず | A(1) 数の表し方の拡張（十進位取り記数法の素地、20までの数） |
| なんじ・なんじはん | C(1) 時刻の読み方（日常生活の中での時刻） |
| 3つのかずのけいさん | A(2) 加法・減法の意味の拡張（3口の数の計算） |
| たしざん(2) / ひきざん(2) | A(2) 加法・減法（繰り上がりのある加法、繰り下がりのある減法） |
| 大きいかず | A(1) 数の表し方（100までの数、十進位取り記数法） |
| たしざんとひきざん（2けた） | A(2) 加法・減法（2位数と1位数、何十どうしの加減） |
| どちらがながい・おおい・ひろい | C(1) 量の大きさの比較（長さ・かさ・広さの直接比較・間接比較） |
| かたちづくり・いろいろなかたち | B(1) 図形についての理解の基礎（身の回りの立体・平面図形の観察） |

## 4. 年間指導計画（めやす）

**※ 教科書会社の実際の年間指導計画とは一致しません。あくまで module 内の単元順序をもとにした、練習配分の目安です。実際は各学級の授業の進度に合わせて調整してください（本モジュールは既習範囲の定着練習として使う想定です）。**

年間約35週、3学期制（1学期：4〜7月、2学期：9〜12月、3学期：1〜3月）を想定した目安配分です。

| 学期 | 週 | 単元 |
|---|---|---|
| 1学期（4〜7月・約14週） | 1〜2週 | 10までのかず |
| | 3週 | なんばんめ |
| | 4〜5週 | いくつといくつ |
| | 6〜8週 | たしざん(1) |
| | 9〜11週 | ひきざん(1) |
| | 12〜14週 | 20までのかず |
| 2学期（9〜12月・約14週） | 15〜16週 | なんじ・なんじはん |
| | 17〜18週 | 3つのかずのけいさん |
| | 19〜22週 | たしざん(2)（くり上がり） |
| | 23〜26週 | ひきざん(2)（くり下がり） |
| | 27〜28週 | 大きいかず（導入） |
| 3学期（1〜3月・約7週） | 29〜31週 | 大きいかず（続き）・たしざんとひきざん（2けた） |
| | 32〜33週 | どちらがながい・おおい・ひろい |
| | 34〜35週 | かたちづくり・いろいろなかたち |

上記はあくまで練習用の配分めやすであり、各単元の練習量は学級の授業進度に沿って前後させてください。特にくり上がり・くり下がりの計算（たしざん(2)・ひきざん(2)）は定着に時間がかかりやすいため、週配分を柔軟に増減することを推奨します。

## 5. 使い方

- 成績簿（gradebook）の授業割り当て画面から、対象クラスに `sansu1` を割り当ててください。単元を絞りたい場合は、割り当て時に単元チェックボックス（focus_units）で対象単元を選択できます（未選択の場合は全単元が出題対象になります）。
- 割り当て後、児童は各自のペースで問題を解いて練習します。教員による個別の進捗確認は成績簿の結果画面から行えます。

**運用のヒント**
- くり上がり・くり下がりの計算単元（たしざん(2)／ひきざん(2)）は、授業でさくらんぼ計算などの考え方を扱った直後に割り当てると定着効果が高まります。
- 学期末や単元末の振り返りタイミングで、既習単元のみを focus_units で指定して短時間の復習セットとして使うのも有効です。
