# 算数 3年 — 先生向けガイド (module key: sansu3)

## 1. 概要

`sansu3` は、小学3年生の算数を対象としたオリジナル問題演習モジュールです。
かけ算・わり算の基礎から、時こくと時間、長さ・重さ・小数・分数などの量と測定、
図形（円と球、三角形）、ぼうグラフと表の読み取り、□を使った式まで、3年生の
算数で扱う内容を幅広くカバーしています。

問題は単元ごとに用意されたジェネレーターが毎回新しい数値・場面でランダム生成
するため、児童は同じ単元を何度でも反復練習できます。採点結果は成績簿
（グレードブック）に自動的に記録され、単元・設問カテゴリ別の正答状況を教師が
確認できます。

**本モジュールはあくまで授業の補助・定着用の教材であり、教室での指導に
取って代わるものではありません。** 新出概念の導入や考え方の指導は教師が
行い、本モジュールは習得した内容の反復練習・定着確認の場としてご活用
ください。

## 2. 単元一覧

`sansu3` には以下の17単元があります（`units.js` に定義された単元構成）。

| No. | 単元名 |
|---|---|
| 1 | かけ算（九九の表とかけ算） |
| 2 | 時こくと時間 |
| 3 | わり算 |
| 4 | たし算とひき算の筆算 |
| 5 | 長さ |
| 6 | ぼうグラフと表 |
| 7 | あまりのあるわり算 |
| 8 | かけ算の筆算（×1けた） |
| 9 | 大きい数のしくみ |
| 10 | 円と球 |
| 11 | 小数 |
| 12 | 重さ |
| 13 | 分数 |
| 14 | □を使った式 |
| 15 | かけ算の筆算（×2けた） |
| 16 | 三角形 |

各単元は1〜2種類の練習セクション（1セクションあたり8〜12問）で構成されて
おり、単元17「3年のまとめ」は他の16単元すべてから出題する総合復習セクション
です。

## 4. 年間指導計画（めやす）

日本の標準的な年間授業週数（約35週、3学期制）を前提とした、単元の
自然な指導順序に沿ったペース配分の目安です。**実際の運用は、各クラスの
教科書の進度に合わせて調整してください。** 本モジュールは授業の進度を
追いかける形で使うことを想定しており、下表は目安の週数配分です。

| 学期 | 期間 | 単元（週数のめやす） |
|---|---|---|
| 1学期 | 4月〜7月（約15週） | 1. かけ算（2週）／2. 時こくと時間（2週）／3. わり算（2週）／4. たし算とひき算の筆算（2週）／5. 長さ（2週）／6. ぼうグラフと表（2週）／7. あまりのあるわり算（2週）／8. かけ算の筆算×1けた（1週） |
| 2学期 | 9月〜12月（約14週） | 8. かけ算の筆算×1けた（続き・1週）／9. 大きい数のしくみ（2週）／10. 円と球（2週）／11. 小数（2週）／12. 重さ（2週）／13. 分数（2週）／14. □を使った式（2週）／15. かけ算の筆算×2けた（1週） |
| 3学期 | 1月〜3月（約6週） | 15. かけ算の筆算×2けた（続き・1週）／16. 三角形（2週）／17. 3年のまとめ（3週・総復習） |

各単元の週内では、授業で新しい内容を学んだ直後にモジュールで練習させる
運用が最も効果的です。単元17「3年のまとめ」は学年末の総復習期間に
継続的に割り当てることを想定しています。

## 5. 使い方

- 成績簿（`hub/gradebook`）の「モジュール割り当て」機能から、担当クラスに
  `sansu3` を割り当てられます。授業の進度に合わせて特定の単元を
  「今週の単元」として指定すると、児童の画面でその単元が優先的に
  表示されます（他の単元も引き続き選択可能です）。
- 割り当て後は、児童が各自のペースで自由に練習できます。同じセクションを
  何度でも繰り返し、その都度新しい問題が出題されます。
- **運用のヒント：**
  1. 授業で新出単元を教えたその週のうちに該当単元を割り当てると、
     定着効果が高まります。
  2. 成績簿の設問カテゴリ別の正答状況を確認し、つまずきが多いカテゴリ
    （例：「あまりを考える文章題」「小数のたし算とひき算」など）が
     見えたら、その部分を重点的に補習してください。
