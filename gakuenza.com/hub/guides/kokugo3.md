# 国語 3年 — 先生向けガイド（module key: kokugo3）

## 1. 概要

`kokugo3` は、小学3年生の国語科（漢字・読解・言葉のきまり）を対象とした、
生徒が自分のペースで取り組める演習モジュールです。次の3つのモードで構成
されています。

- **漢字れんしゅう** — 学年配当漢字200字（学習指導要領 別表準拠）から、
  読み方・漢字えらび・画数などの問題を自動生成して出題します。
- **読解れんしゅう** — 国語の教科書で実際に学習する物語文・説明文7作品に
  対応した読解問題です。**本文そのものは一切含まれておらず**、教室で本文
  を読んだあとに取り組む前提の内容理解確認問題です（著作権上の配慮）。
- **ことば・文ぽう** — こそあど言葉・修飾語・ことわざ／故事成語・ローマ字
  の4分野を、手続き的に生成される問題で練習します。

**このモジュールは授業の補助・定着用の教材であり、国語科の授業そのものを
置き換えるものではありません。** 読解問題は教室での本文指導を前提として
おり、教師による本文の読解指導・言語活動と組み合わせて使うことを想定して
います。

## 3. 学習指導要領との対応（めやす）

> **※ この対応表は「めやす」であり、国語科カリキュラム専門家による確認前
> のドラフトです。実際の年間指導計画・評価規準の作成にあたっては、必ず
> 学校の教育課程・教科書会社の年間指導計画と照合してください。**

| 単元 | 学習指導要領上の主な内容・目標（第3学年及び第4学年） |
|---|---|
| `kanji`（漢字） | 〔知識及び技能〕(1)エ：第3学年に配当されている漢字を読み、漸次書き、文や文章の中で使うこと。 |
| `haru`／`maigo`／`chiichan`／`touge`／`mochimochi`（物語文） | 〔思考力，判断力，表現力等〕C 読むこと：登場人物の行動や気持ちの変化を、場面の移り変わりと結び付けて具体的に想像すること／登場人物の気持ちの変化や性格、情景について、場面の移り変わりと結び付けて具体的に想像すること。 |
| `daizu`／`ari`（説明文） | 〔思考力，判断力，表現力等〕C 読むこと：段落相互の関係に着目しながら、考えとそれを支える理由や事例との関係などについて、叙述を基に捉えること（「問い」と「答え」の文章構成の理解を含む）。 |
| `kosoado`（こそあど言葉） | 〔知識及び技能〕(1)カ：指示する語句（こそあど言葉）と接続する語句の役割について理解すること。 |
| `shuushoku`（修飾語） | 〔知識及び技能〕(1)カ：主語と述語の関係、修飾と被修飾の関係など、文の構成について理解すること。 |
| `kotowaza`（ことわざ・故事成語） | 〔知識及び技能〕(3)イ：長く親しまれていることわざや慣用句、故事成語などの意味を知り、使うこと。 |
| `romaji`（ローマ字） | 〔知識及び技能〕(1)ウ：日常使われている簡単な単語について、ローマ字で表記されたものを読み、ローマ字で書くこと。 |

## 4. 年間指導計画（めやす）

日本の3学期制（年間約35週）を想定した、単元配当の一例です。**教室での本文
指導・実際の教科書進度に合わせて、各校の年間指導計画に沿って調整してくだ
さい。** 漢字とことば・文ぽうの4分野は通年で並行して継続的に取り組むこと
を想定しています。

| 学期 | 時期 | 主な読解単元 | 週あたりのめやす |
|---|---|---|---|
| 1学期 | 4月〜7月（約14週） | `haru`（4月）→ `maigo`（6月） | 読解1単元につき2〜3週。並行して `kanji` を週1〜2回、`kosoado`・`shuushoku` を導入 |
| 2学期 | 9月〜12月（約14週） | `chiichan`（9月）→ `daizu`（11月）→ `touge`（12月） | 読解1単元につき2〜3週。`kanji` を継続、`kotowaza` を追加 |
| 3学期 | 1月〜3月（約7週） | `ari`（1月）→ `mochimochi`（3月） | 読解1単元につき2〜3週。`romaji`・総復習として `kanji` を重点的に |

各読解単元は、教科書でその単元を学習した週〜翌週に割り当てるのが自然です。
上記の週数はあくまで目安であり、学級の教科書進度が優先されます。

## 5. 使い方

1. **成績簿（ハブの gradebook／assign 画面）からクラスへ割り当てます。**
   担当クラスを選び、`kokugo3` を割り当てると、生徒はハブから起動できます。
   `focus_units` で今週の単元（例：`haru` や `kanji`）を指定すると、生徒の
   画面でその単元に「今週」バッジが表示され、優先的に取り組めます。
2. 生徒は漢字・読解・ことばの3モードから自分で選び、自分のペースで演習し
   ます。結果は自動的に成績簿に記録されます。
3. **運用のヒント**
   - 読解単元は、教室で本文を読み終えた直後〜翌週に割り当てると、記憶が
     新しいうちに定着確認ができます。
   - 漢字とことば・文ぽうは出題がランダム生成されるため、同じ単元を複数回
     割り当てても毎回異なる問題に取り組めます。宿題や隙間時間の反復練習に
     向いています。
