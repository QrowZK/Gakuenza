# 外国語（単語） 高学年 — 先生向けガイド (module key: nhvocab)

## 1. 概要

`nhvocab`（New Horizons Vocabulary Practice App）は、小学校高学年
（5・6年生）向けの外国語（英語）**単語練習**モジュールです。
「New Horizon Picture Dictionary」に準拠した約560語の英単語・表現
を収録し、以下の4つの練習モードを提供します。

- **カテゴリー別** — テーマ（動物・食べ物・スポーツなど）ごとに練習
- **5年生 Units** — 5年生の学習順に対応した8つのUnit別に練習
- **6年生 Units** — 6年生の学習順に対応した8つのUnit別に練習
- **ローマ字** — ひらがな→ヘボン式ローマ字の対応練習（英語学習の
  補助として、氏名や地名のローマ字表記に慣れるための練習）

各単語の練習は「英語→日本語」「日本語→英語」の4択クイズ、または
フラッシュカードで行い、英単語には発音を聞ける音声ボタンが付いて
います。

**本モジュールは授業の補助・定着用の教材であり、授業そのものの代わ
りにはなりません。** 教室での聞く・話す・やりとりの指導を前提とし、
すでに学んだ、またはこれから学ぶ単語を、児童が自分のペースで反復
練習し、語彙の定着を図るためのツールとしてご利用ください。

た形で練習できます。

## 3. 学習指導要領との対応（めやす）

> **※ この対応表はドラフトの「めやす」です。カリキュラム担当者の
> 確認前の暫定版であり、正式な整合性確認が完了するまでは参考情報
> としてご利用ください。**

本モジュールは特定学年の1教材ではなく、高学年の外国語科で扱う語彙
を横断的に練習するものであるため、学習指導要領・外国語（高学年）
の「語彙」「表現」に関する目安と対応させています。

| 単元群 | 学習指導要領・外国語（高学年）内容との対応（めやす） |
|---|---|
| 気持ち・数・色・形・様子を表す語 | 〔知識及び技能〕身近で簡単な事柄について、基本的な語彙・表現を理解し、活用する |
| スポーツ・動物・食べ物・飲み物 等（身の回りの事物） | 〔知識及び技能〕日常生活に関する身近で簡単な事柄を表す語彙、〔思考・判断・表現等〕好み・経験を伝え合う |
| 人・家族・性格 | 〔思考・判断・表現等〕自分や身近な人について、考えや気持ちを伝え合う |
| 町・建物・一日の生活・頻度表現 | 〔思考・判断・表現等〕日課や道案内など、身近な暮らしに関する情報を伝え合う |
| 動作の過去形（went / ate 等） | 〔知識及び技能〕過去の出来事を表す基本的な表現を理解する、〔思考・判断・表現等〕経験したことを伝え合う |
| 学校行事・年中行事 | 〔思考・判断・表現等〕学校生活や行事について、感想を交えて伝え合う |
| 職業・部活動 | 〔思考・判断・表現等〕将来の夢や興味のあることについて伝え合う |
| ローマ字練習 | 国語科のローマ字指導内容と関連（外国語科の学習指導要領外の補助的内容） |

## 4. 年間指導計画（めやす）

日本の年間約35週・3学期制（1学期：4〜7月、2学期：9〜12月、3学期：
1〜3月）を前提とした、学年別Unitの指導順に沿う配分の一例です。
**本モジュールはカテゴリー別にも自由に練習できるため、実際の配分
は、クラスの教科書・授業の進度に合わせて調整してください。**

### 5年生

| 学期 | Unit | めやす週数 |
|---|---|---|
| 1学期（4〜7月） | Unit 1（気持ち・数・色・形・教科） | 3〜4週 |
| 1学期（4〜7月） | Unit 2（月・季節・行事・文房具） | 3週 |
| 1学期（4〜7月） | Unit 3（スポーツ・動物・楽器） | 3〜4週 |
| 2学期（9〜12月） | Unit 4（人・家族・性格） | 2〜3週 |
| 2学期（9〜12月） | Unit 5（町・建物・乗り物） | 2〜3週 |
| 2学期（9〜12月） | Unit 6（食べ物・飲み物・デザート） | 3週 |
| 3学期（1〜3月） | Unit 8（職業）＋全Unit共通語の総復習 | 3〜4週 |

### 6年生

| 学期 | Unit | めやす週数 |
|---|---|---|

※ ローマ字練習はUnitに関係なく通年で随時利用でき、隙間時間の練習
に適しています。

## 5. 使い方

1. **成績簿（がくえん座 Hub）から、担当クラスに
   `New Horizons Vocabulary Practice App` を割り当ててください。**
   割り当て後、児童は各自のアカウントでログインし、カテゴリー別・
   学年別Unitの中から自分のペースで練習できます。
2. 児童の解答結果（英語→日本語・日本語→英語クイズ）は自動的に成績
   簿へ記録されます。フラッシュカードは自己学習用のため成績には
   記録されません。

**運用のヒント**
- 授業でその週に学んだ単語のカテゴリー・Unitを指定して家庭学習の
  課題にすると、教室での学習と練習内容がつながりやすくなります。
- 定着が浅い児童には、まず「フラッシュカード」で単語と発音に慣れ
  させてから、4択クイズに取り組ませると効果的です。
