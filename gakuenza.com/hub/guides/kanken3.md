# 漢字検定 3級 — 先生向けガイド (module key: kanken3)

## 1. 概要

`kanken3` は、日本漢字能力検定（漢検）3級の出題範囲を土台にした、漢字・語彙の自動生成型ドリルモジュールです。過去問そのものは一切使わず、協会公表の配当漢字データと本モジュール独自に書き起こした語彙・四字熟語・文例をもとに、実行のたびに新しい問題を生成します（固定の問題バンクではないため、繰り返し取り組んでも問題が尽きません）。

- **教科**：国語（漢字・語彙）
- **想定対象**：漢検3級は協会の目安で「中学校卒業程度」にあたり、小学校の学年別漢字配当を超える範囲を含みます。標準的な学年配当の定着がひと通り済んだ児童や、漢字・語彙に強い関心・意欲のある児童向けの**発展・挑戦的な教材**として位置づけてください。
- **位置づけ**：本モジュールは授業の**補助・定着用の自習教材**です。漢字の指導そのものを代替するものではなく、教師による直接指導・板書・書き取り練習などと組み合わせて使うことを前提としています。

## 2. 単元一覧

`kanken3` には学年別の「単元」（`units.js`）はなく、漢検の出題形式に沿った **10種類の問題カテゴリ** から出題されます。児童はカテゴリを選んで演習するか、複数カテゴリを混ぜた総合演習（ミックス）に取り組みます。

| # | カテゴリ | 内容 |
|---|---|---|
| 1 | 読み | 熟語の正しい読み方を選ぶ |
| 2 | 書き取り | 読みを示し、正しい漢字表記を選ぶ |
| 3 | 熟語の構成 | 二字熟語の組み立て方（似た意味／反対の意味／修飾／目的語など）を選ぶ |
| 4 | 対義語・類義語 | 熟語の対義語・類義語を選ぶ |
| 5 | 四字熟語 | 四字熟語の空欄に入る漢字を選ぶ |
| 6 | 送り仮名 | 正しい漢字と送り仮名の組み合わせを選ぶ |
| 7 | 同音・同訓異字 | 同じ読みで意味の異なる漢字を文脈から選ぶ |
| 8 | 漢字識別 | 複数の熟語に共通して入る漢字を選ぶ |
| 9 | 誤字訂正 | 文中で誤って使われている漢字を正しい字に直す |
| 10 | 部首 | 漢字の部首を選ぶ |

出題のベースとなる配当漢字は284字（漢検3級で新たに配当される漢字。累計対象漢字1623字のうち、4級までの1339字を除いたもの）です。

## 3. 学習指導要領との対応（めやす・要確認）

> ⚠️ 本セクションは**めやす**であり、正式な教育課程の専門家によるチェックを経たものではありません。指導計画に組み込む前に、学校としての確認をお願いします。

`kanken3` は特定の学年の学習指導要領単元ではなく、**漢検3級の配当漢字・出題技能**に沿った教材です。参考として、各カテゴリが対応する学習指導要領（国語）の内容領域を以下に示します。

| カテゴリ | 対応する内容領域（めやす） |
|---|---|
| 読み・書き取り | 〔知識及び技能〕(1)ウ　漢字の読み・書きに関する事項 |
| 部首・漢字識別 | 〔知識及び技能〕(1)ウ　漢字の構成や成り立ちに関する理解 |
| 熟語の構成・同音同訓異字 | 〔知識及び技能〕(1)オ　語句の意味・用法、語彙の質と量に関する事項 |
| 対義語・類義語 | 〔知識及び技能〕(1)オ　語彙を豊かにすること（対義語・類義語の理解） |
| 四字熟語 | 〔知識及び技能〕(1)オ　慣用句・故事成語等、伝統的な言語文化に関する語彙 |
| 誤字訂正 | 〔知識及び技能〕(1)ウ・オ　漢字を正しく使う力、及び文章を推敲する力の基礎 |

なお、扱う漢字の難度（漢検3級配当＝中学校卒業程度）は標準的な小学校の学年別漢字配当表を超えるため、学習指導要領上の学年別漢字とは**1対1に対応しません**。あくまで「発展的な語彙・漢字運用力の育成」という位置づけで参照してください。

## 4. 年間指導計画（めやす）

3学期制・年間約35週を想定した配分例です。児童の実際の定着状況や学級の進度に合わせて調整してください（本表は固定のカリキュラムではなく、練習の目安です）。

| 学期 | 週 | 主なカテゴリ | ねらい |
|---|---|---|---|
| 1学期（4〜7月・約15週） | 第1〜3週 | 読み | 配当漢字の読みに慣れる |
| | 第4〜6週 | 書き取り | 正しい字形での書き取り定着 |
| | 第7〜8週 | 部首 | 部首の理解を通じた字形の整理 |
| | 第9〜10週 | 送り仮名 | 送り仮名の正確な運用 |
| | 第11〜13週 | 熟語の構成 | 熟語の意味構造の理解 |
| | 第14〜15週 | 復習（ミックス） | 1学期範囲の総合確認 |
| 2学期（9〜12月・約13週） | 第16〜18週 | 対義語・類義語 | 語彙の広がりと関連づけ |
| | 第19〜21週 | 同音・同訓異字 | 文脈による漢字の使い分け |
| | 第22〜24週 | 漢字識別 | 共通する漢字の運用力 |
| | 第25〜26週 | 誤字訂正 | 推敲・校正の力 |
| | 第27〜28週 | 復習（ミックス） | 2学期範囲の総合確認 |
| 3学期（1〜3月・約7週） | 第29〜31週 | 四字熟語 | 伝統的な言語文化の語彙 |
| | 第32〜34週 | 総合演習（ミックス） | 全カテゴリの横断的な仕上げ |
| | 第35週 | まとめ・振り返り | 年間の定着状況の確認 |

※ この配分は年間を通じて練習を分散させるための一例です。学級の授業進度や漢字学習の実態に合わせて、各カテゴリの週数や順序は柔軟に変更してください。

## 5. 使い方

1. **成績簿（学級詳細／課題割り当て画面）からクラスに割り当てる。** `hub/admin/class-detail.html` または `hub/gradebook/assign.html` から、対象クラスに `kanken3` を割り当ててください。`kanken3` は学年別の単元選択（フォーカスユニット）を持たないため、割り当て後は児童が自分のペースで全カテゴリ・ミックス演習に取り組めます。
2. **児童は自分のペースで演習する。** 自動生成問題のため、同じカテゴリを繰り返し練習しても問題文が尽きません。休み時間や家庭学習の時間にも活用できます。

**運用のコツ**
- 学期の前半はカテゴリを絞って基礎（読み・書き取り・部首など）を固め、後半はミックス演習で総合力を確認する、という流れが取り組みやすいです。
- 配当漢字が学年配当を超える発展的内容のため、対象を「学級全員」ではなく「意欲・関心の高い児童向けの選択課題」として割り当てる運用も検討してください。
